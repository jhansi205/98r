CSS
casacading style sheet.It is used for describing the look and formatting of a document written in HTML

there are 3 types of css

Inline CSS
Applied directly to the HTML element using the style attribute.example:style="color: blue; font-size: 16px;"
Internal CSS
Defined within the < style > element in the head section of an HTML document. Applies styles to the entire document or specific sections.
<head> <style> p { color: red; font-size: 18px; } </style> </head>
External CSS
Stored in a separate CSS file and linked to the HTML document using the <link> element. Promotes the separation of content and presentation.
example:<link rel="stylesheet" type="text/css" href="styles.css">
p { color: green; font-size: 20px; }
Selector
, selectors are used to target and style specific HTML elements. There are several types of selectors, and two commonly used ones are the element selector and the ID selector, while the class selector is also widely used.
Element Selector
The element selector targets all instances of a specific HTML element and applies the defined styles to them.example:Syntax: elementname { styles } Example: p { color: blue; }
ID Selector:
The ID selector targets a specific HTML element with a unique ID attribute. IDs must be unique within a page. Syntax: #idname { styles } Example: #header { background-color: gray; }
Class Selector:
The class selector targets HTML elements with a specific class attribute. Unlike IDs, multiple elements can share the same class.
