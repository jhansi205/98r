meta tag
Meta tags in HTML (Hypertext Markup Language) are elements that provide metadata about a document. They are included in the head section of an HTML document and do not produce any visible content on the web page. Instead, meta tags convey information to browsers, search engines, and other web services about the document. Here are some common meta tags and their purposes
<meta charset="UTF-8">
Specifies the character encoding for the HTML document. UTF-8 is widely used and supports a broad range of characters.
<meta name="description" content="Brief description of the page">
Provides a concise description of the web page. Search engines may use this information in search results.
<meta name="keywords" content="keyword1, keyword2, keyword3">
Specifies a comma-separated list of keywords related to the content of the page. This tag is less important for search engines today, as they primarily analyze content for relevance.
<meta name="author" content="Author Name">
Indicates the author of the web page.
<eta http-equiv="refresh" content="5;url=https://example.com">
The refresh meta tag is used to automatically refresh or redirect a web page after a specified number of seconds.This example refreshes the page after 5 seconds and redirects to "https://example.com."
